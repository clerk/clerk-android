OrganizationSwitcherDisplayMode

sealed class OrganizationSwitcherDisplayMode

Controls how OrganizationSwitcher renders its trigger.

Normal shows the active account avatar, account name, and chevron. Compact shows only the avatar-sized account trigger. Use normal or compact when a custom trigger avatar size is needed.
