# passwordEnabled



[androidJvm]\




@SerialName(value = &quot;password_enabled&quot;)



val [passwordEnabled](password-enabled.html): [Boolean](https://kotlinlang.org/api/latest/jvm/stdlib/kotlin-stdlib/kotlin/-boolean/index.html)
