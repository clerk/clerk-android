@Serializable
data class SignIn(val id: String, val status: SignIn.Status, val supportedIdentifiers: List<String>? = null, val identifier: String? = null, val supportedFirstFactors: List<Factor>? = null, val supportedSecondFactors: List<Factor>? = null, val firstFactorVerification: Verification? = null, val secondFactorVerification: Verification? = null, val userData: SignIn.UserData? = null, val createdSessionId: String? = null)

The SignIn object holds the state of the current sign-in process and provides helper methods to navigate and complete the sign-in lifecycle. This includes managing the first and second factor verifications, as well as creating a new session.
