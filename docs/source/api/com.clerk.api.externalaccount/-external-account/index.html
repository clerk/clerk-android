ExternalAccount

@Serializable
data class ExternalAccount(val id: String, val identificationId: String, val provider: String, val providerUserId: String, val emailAddress: String, val approvedScopes: String, val firstName: String? = null, val lastName: String? = null, val imageUrl: String? = null, val username: String? = null, val label: String? = null, val verification: Verification? = null, val createdAt: Long)

The ExternalAccount object is a model around an identification obtained by an external provider (e.g. a social provider such as Google).

External account must be verified, so that you can make sure they can be assigned to their rightful owners. The ExternalAccount object holds all necessary state around the verification process.
